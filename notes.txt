Protocol
- CREATE
  - CREATE VAR NAME DTYPE SHAPE 
  - CREATE REL NAME 
  - CREATE DB NAME
- WRITE
  - WRITE DB:REL "[a, b], c" TOTAL 3 TCP 
    - [a, b, a, b, a, b, c, c, c] 
  - WRITE DB:REL "[a, b], c" TOTAL 3 UDP .... 
    - Errors if "[a, b, a, b, a, b, c, c, c]" > Max datagram size
- READ
  - WRITE DB:REL "[a, b[0,:]], c" UDP SORTED a[0,0] INDEXED [4:10]
    - a[0,0] must be a scalar
- CLOSE
  - CLOSE PORTNUM
- DELETE
  - DELETE VAR NAME 
  - DELETE REL NAME 
  - DELETE DB NAME 
  - DELETE DB:REL // TODO
